# Tags

Tagging system for organizing tasks and notes.

## Status Tags

| Tag | Meaning |
|-----|---------|
| `#active` | Currently being worked on |
| `#blocked` | Waiting on someone or something |
| `#review` | Ready for review or feedback |
| `#done` | Completed |
| `#archived` | No longer relevant |

## Priority Tags

| Tag | Meaning |
|-----|---------|
| `#p0` | Critical — do today |
| `#p1` | High priority — this week |
| `#p2` | Nice to have — someday |

## Type Tags

| Tag | Meaning |
|-----|---------|
| `#task` | Single actionable item |
| `#project` | Multi-step initiative |
| `#idea` | Exploratory, not yet committed |
| `#reference` | Reference material, not actionable |

## Area Tags

| Tag | Meaning |
|-----|---------|
| `#portfolio` | Portfolio site work |
| `#ai` | AI side projects |
| `#writing` | Substack / content |
| `#pm` | PM work and career |
